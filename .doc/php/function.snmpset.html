snmpset

(PHP 4, PHP 5)

snmpset — SNMP オブジェクトを設定する

説明

bool snmpset ( string $hostname , string $community , string $object_id , string $type , mixed $value [, int $timeout [, int $retries ]] )

snmpset() 関数は、 object_idで指定した SNMP オブジェクトの 値を設定するために使用します。

パラメータ

hostname

SNMP エージェント。

community

ライトコミュニティ。

object_id

SNMP オブジェクト。

type

value

timeout

最初のタイムアウトまでのマイクロ秒数。

retries

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
