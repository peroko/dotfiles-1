PDO::cubrid_schema

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

PDO::cubrid_schema — 指定したスキーマ情報を取得する

説明

array PDO::cubrid_schema ( int $schema_type [, string $table_name [, string $col_name ]] )

cubrid_schema() 関数を使うと、指定したスキーマ情報をデータベースから取得することができます。 テーブルの情報を知りたい場合は table_name を指定しなければなりません。また、 カラムの情報を知りたければ col_name を指定しなければなりません (PDO::CUBRID_SCH_COL_PRIVILEGE とともにしか使えません)。

パラメータ

schema_type

知りたいスキーマの型。

table_name

スキーマを知りたいテーブル。

col_name

スキーマを知りたいカラム。

返り値

処理に成功すると、スキーマ情報を含む配列を返します。

処理に失敗した場合は FALSE を返します。

例

例1 PDO::cubrid_schema() の例

これは、game テーブルの主キーを取得する方法を示す例です。

<?php
$pk_list = $dbh->cubrid_schema(PDO::CUBRID_SCH_PRIMARY_KEY, "game");
print_r($pk_list);
?>
